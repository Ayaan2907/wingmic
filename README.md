# wingmic

**your social RAM, on disk** — voice-first networking memory

[![License: MIT](https://img.shields.io/badge/License-MIT-FFC452.svg)](LICENSE)
[![Status](https://img.shields.io/badge/status-v0.1--beta-FFC452.svg)](https://wingmic.xyz)

You meet 12 people at a conference. You remember 3 names by day's end. Wingmic is the tool you open right after you meet someone — speak the context, the graph remembers.

`v0.1.1` ships voice capture + recall. Imports (v0.2) and Acts agent (v0.3) on the roadmap. Visit [wingmic.xyz](https://wingmic.xyz) to read more or join the waitlist.

## getting started

```bash
bun install
bun dev
# → http://localhost:3210
```

Requires Bun 1.1+ and Node 20+.
